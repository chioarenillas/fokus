# FOKUS

## 🧩 Description

FOKUS is a web application built with React and TypeScript designed to streamline task management and productivity tracking. It features an interactive dashboard that visualizes task statistics (Total, Pending, In Progress, Completed) and allows users to navigate directly to filtered task lists. 

The project focuses on delivering a clean, type-safe, and responsive interface using modern React practices. It emphasizes efficient state management, dynamic routing with query parameters for filtering, and a modular architecture without relying on heavy UI frameworks.

## 🚀 Live Demo

https://fokus.vercel.app/

## 🚀 Technologies

* HTML5
* CSS3
* TypeScript
* React
* React Router DOM
* Firebase

## 🎯 Features

* Interactive dashboard with real-time statistics cards
* Deep linking from dashboard cards to filtered task views (e.g., clicking "Completed" shows only completed tasks)
* Full CRUD operations for tasks (Create, Read, Update, Delete)
* Status management (Pending, In Progress, Completed)
* SPA navigation with dynamic URL parameters for filtering
* Clean, responsive, and accessible user interface

## 🧠 Key Concepts

* Strong typing with TypeScript interfaces and generics
* Advanced routing patterns with React Router (search params, dynamic segments)
* Complex state management and derived state for statistics
* Custom hooks for reusable logic and data handling
* Component composition and separation of concerns
* Conditional rendering and dynamic UI updates
